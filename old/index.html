Minetest Magic-CTF Website

About »

Magic-CTF is a magical themed capture-the-flag game for the Minetest Game Engine by celeron55. It was written in LUA by Lars and Kai Gerd Müller, alias LMD and KGM, as an alternative to the current one, rubenwardy's CTF, of which the development seems to have stopped.
